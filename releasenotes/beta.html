 The release process for Stone Story is such that new versions are first pushed onto the Beta branch (Steam only). This branch continuously advances with new features until all planned changes for that version are complete and all known bugs are fixed. Then that version is pushed onto the main branch. At that point both branches will be the same, until new features begin development and the process repeats. Changing your game to the Beta branch is a way to help make the game better and experience the new features before everyone else. This comes at the cost of stability, and risks such as losing progress if a critical bug is introduced into the Beta and messes things up. When playing on the Beta you should safeguard your progress by making backups of your save files or at least understand where the game automatically creates backups. For information about save file locations and about switching to the beta branch, go here.
